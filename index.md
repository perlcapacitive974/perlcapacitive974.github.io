## 🛠️ Using the tool

The dashboard shows all installed mods in a list. To add a new mod, use the import button. Select your mod file, usually ending in .fbmod. The tool parses the file and shows it in the list. Check the box next to a mod to activate it. Click the launch button to start your game with the active mods.

### Managing textures and assets
You can swap textures for custom kits. Open the asset browser to view existing files. Select the kit you want to change. Click the replace button to upload your new image file. Ensure your image matches the size of the original file. Save your settings before you exit the tool to apply the changes.

### Working with rosters
The tool includes a roster editor. Open the roster tab to load your current save file. Change player attributes like speed, shot power, or age. Use the search bar to find specific players. Click save to commit your changes to the game database. 

## 📂 Troubleshooting

If the tool does not open, check your security software. Some programs flag new tools as risky. Add an exception for the tool folder in your antivirus settings.

If the game crashes, deactivate your active mods. Start the tool and uncheck the box for each mod. Launch the game again to confirm it runs clean. If the game runs, add one mod at a time to find the conflict.

Always make a backup of your save file before you modify your game. Navigate to your game documents folder and copy your save file to a different folder.

## ⚙️ Advanced configuration
